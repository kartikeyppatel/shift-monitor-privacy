Shift Report Monitor

Shift Report Monitor is an internal automation tool built for a Securitas Inc. security operations team. It automatically reviews daily guard shift report emails, extracts shift activity data from PDF attachments, and compiles a daily summary for the operations team.

The app connects to a single dedicated Gmail account used only to receive these shift report emails. It does not interact with any other Gmail account, and it is not a public or multi-user product — it is a private tool used internally by one team.

Privacy Policy

Last updated: July 2026

What data this app accesses

This app connects to a single dedicated Gmail account (scanthereports@gmail.com) and uses the Gmail API to:

  • Read incoming shift report emails and their PDF attachments
  • Apply and remove Gmail labels to mark emails as processed

How this data is used

Shift report PDF attachments are parsed to extract shift activity data (checkpoint scans, activity logs, incident counts). This data is compiled into an internal spreadsheet and summarized in an email sent only to designated members of the security operations team.

Data sharing

No data accessed by this app is sold, shared, or disclosed to any third party. All processing happens within this single automation and its designated recipients.

Data retention

Processed report data is retained only in the internal spreadsheet maintained by the security operations team.
